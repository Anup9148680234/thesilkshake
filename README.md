## 🚀 Overview

This project is a **modern, animation-first beverage product website** that combines smooth scroll interactions, cinematic transitions, and responsive UI design.

It focuses on delivering a **premium, Apple-level user experience** using GSAP-powered animations and optimized media performance.

---

## ⚙️ Tech Stack

### 🎬 GSAP

Used to power all animations and interactions:

* ScrollTrigger for scroll-based timelines
* SplitText for dynamic text reveals
* Parallax effects for depth
* Scroll-synced video playback
* Pinned sections and timeline orchestration

---

### ⚛️ React

* Component-based architecture
* Reusable UI and animation logic
* Smooth integration with GSAP timelines
* State-driven interactions (carousel, UI states)

---

### 🎨 Tailwind CSS

* Utility-first styling
* Rapid UI development
* Fully responsive layouts
* Clean and maintainable design system

---

### ⚡ Vite

* Instant dev server
* Fast HMR (Hot Module Replacement)
* Optimized production builds
* Minimal configuration

---

## 🔥 Key Features

✨ **Scroll-Driven Storytelling**
Immersive animations synced perfectly with user scroll.

✨ **SplitText Animations**
Engaging text reveals for strong visual impact.

✨ **ScrollTrigger Timelines**
Precise control over animation sequences and timing.

✨ **Parallax Effects**
Adds depth and realism to the UI.

✨ **Pinned Sections**
Keeps sections fixed while animating internal content.

✨ **Scroll-Synced Video**
Video playback controlled by scroll for cinematic feel.

✨ **Custom Animated Carousel**
Interactive product showcase with smooth transitions.

✨ **Image Masking Effects**
Creative transitions using masks and clipping.

✨ **Responsive Animations**
Optimized for mobile, tablet, and desktop.

---

## 🧠 Performance Focus

* Optimized video assets using FFmpeg
* Lightweight animation timelines
* Smooth 60fps interactions
* Efficient scroll handling with GSAP

## 🤸 Quick Start

### 📦 Prerequisites

Make sure you have installed:

* Git
* Node.js
* npm

---

### 🔧 Installation

```bash
npm install
```

---

### ▶️ Run Development Server

```bash
npm run dev
```

Open:
👉 http://localhost:5173
